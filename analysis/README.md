# Matrix Analysis Layer

Python analysis components for the Matrix flash loan arbitrage bot.

## Components

- **oracle/** - Price aggregation and arbitrage detection
- **sati/** - ML models for opportunity prediction
- **persephone/** - Sentiment analysis
- **rama_kandra/** - TVL and liquidity tracking

## Installation

```bash
pip install -e ".[dev]"
```

## Testing

```bash
pytest
```
