## Responsive Breakpoints

```
Desktop:  1440px+ (Full layout as shown)
Laptop:   1024px-1439px (Reduced sidebar, stacked cards)
Tablet:   768px-1023px (Collapsed sidebar, single column)
Mobile:   <768px (Bottom navigation, full-screen views)
```

## Animation Guidelines

- **Transitions**: 150ms ease-out for hover states
- **Page transitions**: 200ms fade
- **Loading states**: Skeleton screens with shimmer animation
- **Charts**: Animate on initial load, update smoothly
- **Notifications**: Slide in from right, auto-dismiss after 5s
- **Real-time updates**: Subtle highlight flash for new data
